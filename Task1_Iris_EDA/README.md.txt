## Objective
To explore and visualize the Iris dataset in order to understand feature distributions, relationships, and potential outliers using exploratory data analysis (EDA).

---

## Dataset Used
- Name: Iris Dataset
- Source: Loaded using seaborn
- Samples: 150
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target Variable: Species

---

## Models Applied
No machine learning models were applied in this task.
This task focuses purely on data exploration and visualization.

---

## Key Results & Findings

- Dataset contains no missing values.
- Petal length and petal width clearly separate species.
- Sepal features show overlapping distributions.
- Some mild outliers were identified through box plots.
- Petal measurements are more discriminative than sepal measurements.
